## Установка

1. Клонировать репозиторий
   ```sh
   git clone https://github.com/Sodbroil/todo-list.git
   ```
2. Установить npm-пакеты
   ```sh
   npm install
   ```
3. Запустить BackEnd
   ```sh
   npm run start:dev
   ```
4. Запустить FrontEnd
   ```sh
   cd frontend
   npm start
   ```
5. Проект запущен.
